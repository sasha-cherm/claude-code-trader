## Key Learnings
1. **KHL markets have STATIC orderbooks** — no dynamic market makers adjust prices during games. Near-res monitoring is useless for KHL. Bid/ask spread is 0.01 but prices don't move.
2. **Indian Super League markets have $0 volume** — completely illiquid, untradeable.
3. **`get_price(token_id, 'buy')` returns the BID, not the ASK** — this is fine for liquid markets (EPL/NBA where bid≈ask) but misleading for illiquid ones. The `place_market_buy()` correctly uses the ask.
4. **Gamma API display prices ≠ CLOB execution prices** — Gamma shows mid/stale prices. CLOB bid is often lower and ask is higher. Pre-game edge calculations must use CLOB prices, not Gamma.
5. **Near-res only works where dynamic MMs exist**: EPL, Bundesliga, Championship, NBA, La Liga — confirmed. KHL, ISL — confirmed NOT working.
